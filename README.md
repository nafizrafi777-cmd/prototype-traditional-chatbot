# Prototype Simple Traditional Chatbot

## Overview

This project demonstrates the implementation of a simple traditional rule-based chatbot using Python. Unlike Large Language Models (LLMs), this chatbot relies on predefined rules and conditional statements to respond to user inputs.

The project was developed for the Artificial Intelligence for Human-Computer Interaction (AI-HCI) course.

## Features

- Rule-based conversation
- Dictionary-based knowledge base
- Interactive command-line interface
- Predefined responses
- Exit command ("bye")

## Technologies Used

- Python 3
- Google Colab
- GitHub

## Example Conversation

User: hello

Chatbot: Hello! Nice to meet you.

User: what is ai

Chatbot: Artificial Intelligence enables computers to perform tasks that usually require human intelligence.

User: bye

Chatbot: Goodbye! Have a wonderful day!
